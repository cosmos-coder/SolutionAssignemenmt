# Patient-Charting-Application <br/>
**JIRA Link :**  https://jira2.cerner.com/browse/DEVACDMY-35640 <br/>
This is Windows Application created Using Visual c++ and The Microsoft Foundation Class (MFC) library which provides a set of functions, constants, data types, and classes to simplify creating applications for the Microsoft Windows operating systems.<br/>With this Application, physician will be able to chart search patients, chart allergies, view allergies charted on any patient.<br/>
### MFC <br/>
Microsoft Foundation Class Library is a C++ object-oriented library for developing desktop applications for Windows.<br/>
